That's it. Linux Autumn 2014 is over. This year I finally decided to give a talk instead of only listening and networking.

The subject of this talk was Orchestrating Docker containers at scale. As this was first time Docker ever appeared on Linux Autumn Conference I decided to explain basics of those containers and later jump into deeper water or orchestration.
